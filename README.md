# CosmicDev | Portfólio Tecnológico

Um portfólio web moderno e interativo com tema espacial, criado com HTML5, CSS3 e JavaScript vanilla.

## 🚀 Demonstração

🌐 **Site ao vivo**: [https://ivaanildo.github.io/cosmic-portfolio/](https://ivaanildo.github.io/cosmic-portfolio/)

## 📋 Características

### Design e Visual
- **Tema Espacial**: Design inspirado no cosmos com cores neon e efeitos visuais
- **Responsivo**: Adaptado para todos os dispositivos (desktop, tablet, mobile)
- **Animações CSS**: Transições suaves e efeitos visuais impressionantes
- **Tipografia Moderna**: Combinação das fontes Montserrat e Orbitron

### Efeitos Interativos
- **Estrelas Animadas**: Campo de estrelas com animação de cintilação
- **Estrelas Cadentes**: Meteoros que atravessam a tela periodicamente
- **Rastro do Mouse**: Cursor personalizado com efeito de rastro luminoso
- **Sistema Solar**: Animação 3D do sol, terra e lua em órbita
- **Partículas Interativas**: Efeitos de partículas ao clicar e mover o mouse

### Funcionalidades
- **Navegação Suave**: Scroll suave entre seções
- **Menu Responsivo**: Menu hamburger para dispositivos móveis
- **Formulário de Contato**: Sistema de validação e notificações
- **Animações no Scroll**: Elementos aparecem conforme o usuário navega
- **Otimizações de Performance**: Lazy loading e controle de animações

## 🚀 Como Usar

### Visualização Simples
1. **Clonar o repositório**:
   ```bash
   git clone https://github.com/Ivaanildo/cosmic-portfolio.git
   cd cosmic-portfolio
   ```

2. **Abrir o projeto**: Simplesmente abra o arquivo `index.html` em um navegador web moderno

### Desenvolvimento Local
1. **Instalar dependências** (opcional):
   ```bash
   npm install
   ```

2. **Iniciar servidor de desenvolvimento**:
   ```bash
   npm run dev
   ```

3. **Navegar**: Use o menu superior ou role a página para explorar as seções
4. **Interagir**: Mova o mouse para ver o efeito de rastro e clique para criar explosões de estrelas
5. **Contato**: Preencha o formulário na seção de contato para enviar mensagens

- Contraste adequado de cores

## 🔧 Personalização

Para personalizar o portfólio:

1. **Cores**: Modifique as variáveis CSS em `css/main.css`
2. **Conteúdo**: Edite o HTML em `index.html`
3. **Animações**: Ajuste os keyframes em `css/animations.css`
4. **Funcionalidades**: Modifique os módulos JavaScript em `js/`

## � Deployment

### GitHub Pages
Este projeto está configurado para deploy automático no GitHub Pages:

1. **Acesse**: [Settings](https://github.com/Ivaanildo/cosmic-portfolio/settings/pages) do repositório
2. **Configure**: Source como "Deploy from a branch" 
3. **Selecione**: Branch `main` e pasta `/ (root)`
4. **Salve**: O site estará disponível em poucos minutos

### Outros Serviços
- **Netlify**: Conecte seu repositório GitHub para deploy automático
- **Vercel**: Importe o projeto do GitHub
- **Surge.sh**: Use `npx surge` para deploy rápido
